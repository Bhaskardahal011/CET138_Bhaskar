3. The CSS Box Model

Every element is a box. From inside to outside: content → padding → border → margin. This card uses:

  • padding: 24px – space inside the border
  • border: 1px solid #dce1e7 – thin grey border
  • border-radius: 8px – rounded corners
  • margin-bottom: 24px – space below the card
  • box-shadow: 0 2px 8px rgba(0,0,0,0.08) – subtle shadow
